# 碳排放管理系统 - PC端高保真原型设计

## 项目概述

这是碳排放管理系统的PC端高保真原型设计，专为桌面端用户优化，提供更强大的数据展示和管理功能。

## 技术栈

- **HTML5** - 页面结构
- **Tailwind CSS** - 样式框架（CDN引入）
- **FontAwesome 6.4** - 图标库
- **Google Fonts** - Inter字体

## 项目结构

```
carbon-management-pc/
├── index.html              # 主入口页面（原型总览）
├── css/
│   └── styles.css          # 公共样式文件
└── pages/
    ├── dashboard.html      # 数据概览
    ├── data-entry.html     # 数据录入
    ├── monitoring.html     # 实时监测
    ├── analysis.html       # 数据分析
    ├── reports.html        # 报告中心
    ├── trading.html        # 碳交易管理
    ├── solutions.html      # 减排方案
    ├── company.html        # 企业管理
    └── profile.html        # 个人中心
```

## 设计特点

### 1. PC端优化布局
- **固定侧边栏** - 240px宽度，左侧固定
- **顶部工具栏** - 64px高度，包含快捷操作
- **多列布局** - 充分利用大屏幕空间
- **响应式设计** - 支持不同分辨率

### 2. 统一的设计语言
- **绿色环保主题** - 与移动端保持一致
- **卡片式设计** - 统一的圆角、阴影
- **清晰的信息层次** - 合理的间距和对齐
- **图标+文字** - 提高可识别性

### 3. 丰富的数据展示
- **大尺寸图表** - 充分利用屏幕空间
- **多维度统计** - 网格布局展示关键指标
- **表格展示** - 历史记录、交易记录等
- **进度可视化** - 进度条、环形图等

### 4. 完善的交互元素
- **面包屑导航** - 页面层级清晰
- **标签页切换** - 功能模块分类
- **筛选和搜索** - 快速查找数据
- **批量操作** - 提高工作效率

## 与移动端的区别

### 布局差异
- **移动端**: 底部导航、单列布局、卡片堆叠
- **PC端**: 侧边导航、多列布局、并排展示

### 功能差异
- **移动端**: 简化操作、核心功能优先
- **PC端**: 完整功能、批量操作、高级筛选

### 展示差异
- **移动端**: 精简展示、关键信息优先
- **PC端**: 详细展示、更多数据维度

## 转换为Figma

本原型设计已考虑Figma转换需求：
1. 使用语义化的HTML结构
2. 样式与结构分离
3. 使用标准CSS类名
4. 图片使用在线资源链接

### 转换步骤
1. 使用HTML to Figma插件导入HTML文件
2. 或使用Figma的Dev Mode手动复制样式
3. 调整图层结构和命名
4. 添加交互原型链接

## 开发建议

1. 保持统一的设计语言
2. 优化大屏幕数据展示
3. 注重键盘操作友好性
4. 支持打印和导出功能
5. 考虑无障碍访问
